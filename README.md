# PowerBI_data_professional_survey

# Project Overview
This project focuses on building an interactive power BI dashboard to analyze data professional survey.

# Dataset
Survey data collected from data professionals across different countries and job roles.

# Tools Used
- Power BI Desktop


# Folder Structure
- data → Raw survey dataset
- pbix → Power BI dashboard file
- output → Dashboard screenshots

# How to View
- Open the `.pbix` file in Power BI Desktop
- View images in the output folder for quick insights
